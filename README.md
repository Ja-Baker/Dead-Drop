## Development

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Start production server
npm start
```

## Deployment to Railway

This app is configured for Railway deployment:

1. **Connect your repo** to Railway
2. Railway will **auto-detect** the configuration from railway.json and nixpacks.toml
3. It will **build** the app using npm run build
4. It will **serve** the static files using the Express server

### Environment Variables

No environment variables needed for MVP demo.
